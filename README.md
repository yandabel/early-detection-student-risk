# Early Warning System for Student Risk Detection

## Description

Cette application utilise l'intelligence artificielle pour détecter précocement les élèves à risque scolaire.

## Technologies

- React.js
- FastAPI
- XGBoost
- SHAP
- Recharts
- Pandas

## Fonctionnalités

- Import Excel
- Prédiction individuelle et batch
- Analytics
- Explainable AI (SHAP)
- Alertes
- Rapports PDF et Excel
